API Reference

Comprehensive documentation for the MCP Marketplace API. Use these endpoints to interact with MCP servers and integrations programmatically.

Authentication

Endpoints for user authentication and token management.

MethodEndpointDescription
POST/api/auth/loginAuthenticate a user and get a token
POST/api/auth/refreshRefresh an authentication token
POST/api/auth/logoutInvalidate a token
Servers

Endpoints for managing MCP servers.

MethodEndpointDescription
GET/api/serversList all available MCP servers
GET/api/servers/{id}Get details for a specific server
POST/api/servers/connectConnect to an MCP server instance
Integrations

Endpoints for working with MCP integrations.

MethodEndpointDescription
GET/api/integrationsList all available integrations
GET/api/integrations/{id}Get details for a specific integration
POST/api/integrations/installInstall an integration to a server
Data

Endpoints for data management and querying.

MethodEndpointDescription
GET/api/data/{server_id}Query data from a server
POST/api/data/{server_id}Send data to a server
DELETE/api/data/{server_id}Clear data from a server
